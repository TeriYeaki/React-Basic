# Beginners React JS Repository

Welcome to my Beginners React JS Repository! This repository is my personal learning journey through the basics of React, using essential hooks and concepts to build a solid foundation with the React Library.

## About

I created this repository as a way to learn and practice React. The content here is inspired by Bro Code's excellent "React Full Course for free ⚛️ (2024)" on YouTube. If you're interested, you can watch the full course [here](https://www.youtube.com/watch?v=CgkZ7MvWUAA&t=176s).

### Course Highlights

This repository covers a range of topics that are fundamental to React development. Here's what you'll find:

1. **Card components**: Building reusable card components.
2. **Add CSS styles**: Applying CSS styles to components.
3. **Props**: Understanding and using props to pass data.
4. **Conditional rendering**: Rendering components conditionally.
5. **Render lists**: Displaying lists of data.
6. **Click events**: Handling user interactions with click events.
7. **useState() hook**: Managing state within components.
8. **onChange event handler**: Handling form inputs and changes.
9. **Color picker app**: Building a simple color picker application.
10. **Updater functions**: Using functions to update state.
11. **Update OBJECTS in state**: Managing state with objects.
12. **Update ARRAYS in state**: Managing state with arrays.
13. **Update ARRAY of OBJECTS in state**: Handling complex state updates.
14. **To-do list app**: Creating a basic to-do list application.
15. **useEffect() hook**: Using the useEffect hook for side effects.
16. **Digital clock app**: Building a digital clock.
17. **useContext() hook**: Using context for state management.
18. **useRef() hook**: Accessing and manipulating DOM elements.
19. **Stopwatch app**: Creating a stopwatch application.

## Features

- Comprehensive introduction to React

- Hands-on approach to learning state management
- Basic routing and styling

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- Node.js
- npm (Node Package Manager)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/TeriYeaki/React-Basic.git
```

2. Navigate to the project directory:

```bash
cd React-Basic
```

3. Install the dependencies:

```bash
npm install

```

### Running the Application

To start the development server, run:

```bash
npm start
```

Open your browser and visit http://localhost:3000 to see the application in action.
